Overview

This module scrapes webpage data using Python and Selenium WebDriver.
It extracts: Webpage Title, Body Text, All Hyperlinks

Requirements

Python 3
Selenium
ChromeDriver
Install Selenium: pip install selenium
Download ChromeDriver (matching your Chrome version): https://chromedriver.chromium.org/downloads


Run the Program
python web_scraper.py
It will prompt:
Enter URL 1:
Enter URL 2:
